# REST API

| Метод   | Endpoint                 | Описание                    |
| ------- | ------------------------ | --------------------------- |
| **GET** | `/search/users?q=...`    | поиск пользователей         |
| **GET** | `/search/servers?q=...`  | поиск серверов              |
| **GET** | `/search/messages?q=...` | поиск сообщений (по тексту) |
